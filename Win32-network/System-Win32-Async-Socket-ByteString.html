send ∷ Socket → ByteString → IO Int Source #

Send a ByteString over a socket, which must be in a connected state, and must be associated with an IO completion port via associateWithIOManager. Returns number of bytes sent.

sendAll ∷ Socket → ByteString → IO () Source #

Like send but makes sure that all bytes are actually sent.

recv ∷ Socket → Int → IO ByteString Source #

Receive a ByteString from a socket, which must be in a connected state, and must be associated with an IO completion port via associateWithIOManager. It may return less bytes than requested.
