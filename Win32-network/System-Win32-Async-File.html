HANDLE API

readHandle ∷ HANDLE → Int → IO ByteString Source #

Read a given number of bytes from a HANDLE. The HANDLE must be opened with fILE_FLAG_OVERLAPPED and associated with an IO completion port via associateWithIOCompletionProt.

writeHandle ∷ HANDLE → ByteString → IO () Source #

Write a ByteString to a HANDLE. The HANDLE must be opened with fILE_FLAG_OVERLAPPED and associated with an IO completion port via associateWithIOCompletionPort.

NamedPipe API

connectNamedPipe ∷ HANDLE → IO () Source #

Connect named pipe aka accept a connection. The HANDLE must be opened with FILE_FLAG_OVERLLAPPED and associated with IO completion port via associateWithIOCompletionPort.
