eRROR_IO_PENDING ∷ ErrCode Source #

For async operations, getLastError reports ERROR_IO_PENDING. It is not an error per se, but used by the os to notify that the i/o will be completed asynchronously. For the scope of this library it means that the IOManager thread will be notified when the operation completes.
